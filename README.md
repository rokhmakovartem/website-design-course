# Мій HTML-сайт (університетський проєкт)

## Про проект
Це мій перший навчальний веб-проект, створений для закріплення базових навичок роботи з HTML5. Проект являє собою невеликий сайт-візитку, що складається з декількох пов'язаних сторінок.

## Що реалізовано
* **Базова структура:** Багатосторінкова архітектура з єдиною навігацією.
* **Семантична та доступна розмітка:** Використання тегів `header`, `main`, `footer`, `nav`, `figure`, а також ARIA-атрибутів для доступності екранами для читання (A11y).
* **Контент:** Робота з таблицями (`table`, `thead`, `tbody`, `caption`) та вбудовування мультимедійного контенту (`video`, `audio`) з резервним текстом.
* **Інтерактивні UI-компоненти:** Перемикач світлої/темної теми зі збереженням вибору в `localStorage`.
  * Акордеон (для блоку FAQ) з плавною анімацією розкриття через `scrollHeight`.
  * Модальне вікно (лайтбокс) для збільшення зображень без "стрибків" макета (компенсація скролбару).
  * Кнопка "Нагору", що з'являється при прокрутці.
* **Динамічна фільтрація:** Фільтр навчальних проєктів за категоріями (HTML, CSS, JS) без перезавантаження сторінки з автоматичним перешикуванням сітки на CSS Grid.
* **Просунута робота з формами:** Кастомна JavaScript-валідація в реальному часі.
  * Динамічний лічильник символів для `textarea`.
  * Автозбереження введених даних (чернетки) у `localStorage`.
  * Перехоплення відправки форми (`e.preventDefault()`) та виведення зібраних результатів за допомогою об'єкта `FormData`.
* **Оптимізація та архітектура:** Єдиний реєстр SVG-іконок у JavaScript для уникнення дублювання коду.
  * Консистентна колірна схема на базі CSS-змінних (`:root`).
  * Повна адаптивність під мобільні пристрої (Responsive Web Design).

## Структура файлів
* `index.html` — Головна сторінка з планом навчання та розкладом.
* `pages/about.html` — Сторінка "Про мене" з відео та аудіо.
* `pages/contact.html` — Сторінка з формою зворотного зв'язку.
* `pages/submit_page.html` — Сторінка успішної відправки даних форми.
* `assets/` — Каталог для зберігання підключених ресурсів (зображень, медіа, скриптів).

## Як запустити
1. Завантажте папку з проектом на свій комп'ютер.
2. Знайдіть у кореневій папці файл `index.html`.
3. Двічі клікніть по ньому, щоб відкрити у будь-якому сучасному веб-браузері (Google Chrome, Firefox, Safari).
4. Ніяких додаткових програм, серверів чи налаштувань для перегляду не потрібно.
